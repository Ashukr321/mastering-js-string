# JavaScript String Concepts

This document covers fundamental concepts of strings in JavaScript.

## Fundamental Topics

### 1. [What is a String?](../00_concepts/what_is_a_string.js)
A primitive data type used to represent textual data.

### 2. [Immutability](../00_concepts/immutability.js)
Strings in JavaScript are immutable; once created, they cannot be changed.

### 3. [Template Literals](../00_concepts/template_literals.js)
Allows for multi-line strings, interpolation, and expressions using backticks (``).

### 4. String Creation
Strings can be created as literals (using '', "", or ``) or as String objects (using `new String()`).
